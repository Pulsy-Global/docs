# Pulsy Documentation

Pulsy is the complete suite for tracking Web3 data. We bring together real-time DeFi feeds, cross-chain intelligence, and historical backfills so your team can monitor, automate, and make decisions in real-time.

## Pulsy Product Suite

### Atria (Beta)
- Create and launch tailored DeFi event feeds in minutes with prebuilt templates.
- Configure parameters, thresholds, and delivery channels across open-source or hosted deployments.
- Ship production-ready feeds with flexible delivery options.

Read more at [pulsy.app/atria](https://pulsy.app/atria).

### XFlow
- Track and analyze liquidity flows across bridges and chains with real-time filtering and search.
- Monitor bridge swaps as they happen, stream them over API/webhooks, and query historical swap activity.
- Unlock cross-chain transaction volume insights for dashboards, operations, and compliance.

Read more at [pulsy.app/xflow](https://pulsy.app/xflow).

### W12
- Run deep historical analysis in hours, not weeks, and save on internal tooling.
- Replay feeds from any block height, schedule replays, and target precise block ranges.
- Export in flexible formats that plug into your existing data stack.

Read more at [pulsy.app/w12](https://pulsy.app/w12).

# Pulsy Documentation

Pulsy is the complete suite for tracking on-chain events. Pulsy brings together event-driven workflows and cross-chain intelligence, giving teams a foundation to turn on-chain events into actions for monitoring, automation, and decision making.

## Pulsy Product Suite

### Atria

Atria is Pulsy's off-chain backend for event-driven blockchain workflows. It helps teams process on-chain data, apply custom logic, and trigger real-time actions across their systems.

- Build feeds that read blockchain data and apply custom logic.
- Use AI-assisted feed creation, custom feeds, and the open-source Atria Library.
- Connect feeds to external systems in real time.
- Run through Atria Cloud, self-managed, private, or on-prem deployments.

Read more at [pulsy.app/atria](https://pulsy.app/atria).

### XFlow

XFlow provides cross-chain intelligence to analyze liquidity flows and bridge activity across chains with real-time and historical visibility.

- Track bridge activity across multiple chains.
- Explore real-time and historical cross-chain flows.
- Build bridge analytics, monitoring infrastructure, dashboards, and data pipelines.
- Use coverage across 70+ integrated bridges.

Read more at [pulsy.app/xflow](https://pulsy.app/xflow).
